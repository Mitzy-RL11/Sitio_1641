5.1 Dominios Web y Subdominios

Un **Dominio Web** es el nombre único y exclusivo que recibe un sitio web en Internet para que un usuario pueda localizarlo sin necesidad de memorizar su dirección IP numérica. El Sistema de Nombres de Dominio (DNS) se encarga de realizar esta traducción.

Dominios de Nivel Superior (gTLD)

Extensiones genéricas globales que definen el propósito del sitio. Ejemplos comunes: .com (comercial), .org (organizaciones), .edu (educación).

Dominios Geográficos (ccTLD)

Extensiones de dos letras reservadas para países o territorios específicos. Ejemplos comunes: .mx (México), .es (España), .co (Colombia).

Un **Subdominio** es una ramificación o extensión del dominio principal creada para organizar diferentes secciones independientes dentro de un mismo sitio web, sin necesidad de adquirir un dominio nuevo (por ejemplo: `blog.tusitio.com` o `tienda.tusitio.com`).

5.2 Certificados de Seguridad (SSL/TLS)

Un certificado **SSL/TLS** (Secure Sockets Layer / Transport Layer Security) es un título digital que autentica la identidad de un sitio web y habilita una conexión cifrada. Su función primordial es proteger los datos sensibles (credenciales, datos bancarios) en tránsito entre el navegador del usuario y el servidor web.

Al instalarse en el servidor web, activa el candado de seguridad en la barra de direcciones y transforma el protocolo de comunicación inseguro HTTP hacia su versión protegida: **HTTPS**.

5.3 URL (Localizador Uniforme de Recursos)

Una **URL** (Uniform Resource Locator) es la dirección específica asignada a cada uno de los recursos disponibles en la red (páginas HTML, imágenes, documentos) para que puedan ser localizados por el navegador.

🔍 Anatomía de una URL Segura

https://www.uneve.edu.mx/multimedia/index.html
• https:// : Protocolo seguro de transferencia de datos.
• www. : Subdominio de red secundaria estándar.
• uneve : Nombre de dominio principal (Identidad).
• .edu.mx : Extensión o TLD (Institucional / México).
• /multimedia/index.html : Ruta de acceso al archivo dentro de public_html.
