2.1 COMPRESIÓN DE DATOS

Modulo: Algoritmos // Status: Definición

OPTIMIZACIÓN: ACTIVA

La compresión de datos es el proceso tecnológico fundamental mediante el cual se reduce el tamaño físico de un bloque de información. Este procedimiento consiste en codificar la información utilizando menos bits que la representación original.

El objetivo principal es doble: ahorrar espacio de almacenamiento en discos duros y servidores, y disminuir el tiempo necesario para transmitir esos datos a través de redes como Internet.

Sin la compresión, la web moderna tal como la conocemos no existiría. Servicios como el streaming de video en 4K, la música en la nube o la descarga rápida de aplicaciones dependen enteramente de nuestra capacidad para "encoger" los datos antes de enviarlos y "reconstruirlos" al llegar a su destino.
