Infraestructura de Software

Los servidores web son los encargados de despachar el contenido a los usuarios finales. Dependiendo de la tecnología, ofrecen distintos niveles de estabilidad y escalabilidad.

🌐

Apache / Nginx

Estándares de la industria para servir contenido estático y dinámico con alta estabilidad.

💻

Servidores Locales

Entornos controlados (XAMPP / Docker) para realizar pruebas críticas antes del despliegue.

☁️

Cloud Servers

Instancias virtuales escalables que permiten ajustar recursos según la demanda en tiempo real.

Componentes de Hardware

  • 🧠 CPU: Procesadores multinúcleo para tareas paralelas.
  • ⚡ RAM: Memoria de alta velocidad para peticiones simultáneas.
  • 💾 NVMe: Almacenamiento ultra rápido para reducir latencia.

Stack de Software

El sistema operativo (comúnmente Linux) actúa como la base sobre la cual se ejecutan bases de datos (MySQL) y lenguajes de backend.

Stack Tecnológico

Forma de Trabajo: Request-Response

1. Petición (Request): El cliente solicita datos vía protocolo HTTP.
2. Procesamiento: El servidor procesa la lógica y consulta la base de datos.
3. Respuesta (Response): Se envían los archivos finales al navegador del usuario.
