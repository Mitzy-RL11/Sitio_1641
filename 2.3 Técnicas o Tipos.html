2.3 TIPOS DE COMPRESIÓN

Modulo: Algoritmos // Status: Clasificación

Aunque existen cientos de algoritmos diferentes (fórmulas matemáticas para comprimir), todos se clasifican en dos grandes familias o técnicas fundamentales. La elección entre una y otra depende del uso final de los datos y de si podemos tolerar algún cambio en la información.

LOSSLESS

(Sin Pérdida)

Permite reconstruir el archivo original exactamente bit por bit. No se pierde ni un solo dato. Es esencial para textos, programas informáticos y bases de datos.

LOSSY

(Con Pérdida)

Elimina información para lograr archivos muy pequeños. La reconstrucción es una aproximación, no una copia exacta. Se usa en multimedia (fotos, música, video).
